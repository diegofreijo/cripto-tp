Aqui hay un ejemplo para probar que tiene mas de una solucion:
 
$maximizar   x1 + 3x2 - x3      $
$sujeto a   2x1 + 2x2 - x3 <= 10$
$           3x1 - 2x2 + x3 <= 10$
$            x1 - 3x2 + x3 <= 10$
                    
              $  x1,x2,x3 >= 0  $
 
Solución: probar con $ x1 = 0 ; x2 = 5+0.5t ; x3 = t.$
